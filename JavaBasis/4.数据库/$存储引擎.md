##  存储引擎

### 主要类型

显示引擎类型：show engines;

修改引擎类型：alter table 表名 engine = 引擎类型

> - InnoDB、MyISAM、Memory（*主要*）
>   1. InnoDB：支持事务、外键；**行锁**
>   2. MyISAM：不支持事务、外键；添加速度快，**表锁**
>   3. Memory：存储在内存中【关闭服务后数据丢失，表结构依然存在】、执行速度快、默认哈希索引，**表锁**
> - CSV、Archive、Mrg_memory

###  特点

|      特点      | MyISAM | InnoDB |    Memory    | Archive |
| :------------: | :----: | :----: | :----------: | :-----: |
| *批量插入速度* |   高   |   低   |      高      |  极高   |
|   *事务安全*   |        |  支持  |              |         |
|    全文索引    |  支持  |        |              |         |
|    *锁机制*    |  表级  |  行级  |     表级     |  行级   |
|   *存储限制*   |   无   |  64TB  |      有      |   无    |
|    B树索引     |  支持  |  支持  |     支持     |         |
|    哈希索引    |        |  支持  |     支持     |         |
|    集群索引    |        |  支持  |              |         |
|    数据缓存    |        |  支持  |     支持     |         |
|    索引缓存    |  支持  |  支持  |     支持     |         |
|   数据可压缩   |  支持  |        |              |  支持   |
|   *空间使用*   |   低   |   高   | 由数据量决定 |  极低   |
|    内存使用    |   低   |   高   |      中      |   低    |
|     *外键*     |        |  支持  |              |         |
